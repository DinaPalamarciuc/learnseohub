SEO Glossary

28 essential terms every SEO professional should know

AI Mode
Google's generative AI search interface that synthesizes multi-source answers.
AEO
Answer Engine Optimization — optimizing for Featured Snippets and direct answers.
Backlink
A link from another website pointing to yours. A key ranking signal.
Core Web Vitals
Google's UX metrics: LCP (loading), INP (interactivity), CLS (stability).
Crawl Budget
The number of pages Google will crawl on your site in a given timeframe.
E-E-A-T
Experience, Expertise, Authoritativeness, Trust — Google's content quality framework.
Entity
A distinct concept (person, place, thing) that search engines can understand and connect.
Featured Snippet
Answer box shown above organic results. Also called Position Zero.
GEO
Generative Engine Optimization — optimizing content for AI-powered search engines.
Google Business Profile
Free listing for local businesses in Google Search and Maps.
Hreflang
HTML attribute telling Google which language/region a page targets.
Index Bloat
When search engines index low-value pages, diluting your site's quality signals.
JSON-LD
JavaScript Object Notation for Linked Data — preferred format for Schema markup.
Knowledge Graph
Google's database of entities and their relationships.
LLM
Large Language Model — AI systems like GPT, Gemini, Claude that power AI search.
Long-Tail Keyword
Specific 3+ word phrases with lower volume but higher conversion intent.
NAP
Name, Address, Phone — must be consistent across all local citations.
Orphan Page
A page with no internal links pointing to it. Hard for crawlers to discover.
PageRank
Google's original algorithm measuring page importance via link equity.
RAG
Retrieval-Augmented Generation — how AI search engines find and cite sources.
robots.txt
File telling crawlers which pages to crawl or avoid on your site.
Schema Markup
Structured data vocabulary (schema.org) helping search engines understand content.
Search Intent
The purpose behind a search query: informational, navigational, commercial, transactional.
SERP
Search Engine Results Page — what you see after searching on Google.
Sitemap
XML file listing all important URLs on your site for search engines.
Topical Authority
Deep, comprehensive coverage of a topic that signals expertise to Google.
UCP
Universal Commerce Protocol — Google's standard for AI agent-powered purchases.
Zero-Click
Search where the answer appears directly in SERP — user doesn't click through.
